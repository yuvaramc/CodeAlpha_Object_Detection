# 🎯 AI-Based Object Detection and Tracking using YOLOv8

This project is developed as part of the **CodeAlpha Artificial Intelligence Internship**. It demonstrates real-time object detection and tracking in video streams using advanced deep learning techniques.

---

## 📌 Project Overview

This application uses **YOLOv8 (You Only Look Once)** to detect multiple objects in uploaded videos. The model processes each frame, identifies objects, and displays bounding boxes with labels for accurate real-time detection and tracking.

The primary objective of this project is to explore how modern computer vision techniques can be applied to real-world scenarios such as surveillance, traffic monitoring, and smart video analytics.

---

## 🚀 Key Features

- Real-time object detection using YOLOv8
- Multi-object tracking
- Video upload support (MP4, AVI, MOV)
- Accurate object localization with bounding boxes
- Interactive Streamlit web interface
- Fast and efficient AI-powered detection

## ▶️ How to Run This Project

### Step 1: Install the required libraries

```bash
pip install -r requirements.txt
```

### Step 2: Run the application

```bash
python -m streamlit run app.py
```

### Step 3: Use the application

1. Open the Streamlit application in your browser.
2. Click **Upload a Video**.
3. Select an MP4, AVI, or MOV file from your computer.
4. Wait for the model to process the video.
5. View the detected objects with bounding boxes in real time.

---

## 📸 Output

The application detects and tracks objects in the uploaded video, displaying bounding boxes and labels for each detected object through an interactive Streamlit interface.
