# 🎯 AI-Based Object Detection and Tracking using YOLOv8

This project is developed as part of the **CodeAlpha Artificial Intelligence Internship**. It demonstrates real-time object detection and tracking in video streams using advanced deep learning techniques.

---

## 📌 Project Overview

This application uses **YOLOv8 (You Only Look Once)** model to detect multiple objects in a video. The system processes each frame, identifies objects, and draws bounding boxes with labels, enabling real-time visual tracking.

The goal of this project is to understand how modern computer vision models work in practical applications such as surveillance, traffic monitoring, and smart analytics.

---

## 🚀 Key Features

- Real-time object detection using YOLOv8
- Multi-object tracking with unique identification
- Video upload support (MP4, AVI, MOV formats)
- High-speed inference with accurate predictions
- Interactive web interface using Streamlit
